## Project Name : DigiTools Platform


##Description

DigiTools Platform is a modern web-based collection of useful digital tools designed to simplify everyday tasks. It provides multiple utilities in a single platform with a clean UI and smooth user experience. The goal of this project is to make productivity tools easily accessible for everyone.

##Technologies Used
 1. React.js
2.  Vite
3. Tailwind CSS
 4.Netlify (Deployment via Netlify)
 live link : https://digitools-platform-m2.netlify.app/

## Features
🔹 1. Multiple Tools in One Place

Access different digital utilities from a single platform without switching websites.

🔹 2. Fast & Responsive UI

Built with modern technologies to ensure smooth performance and mobile-friendly design.

🔹 3. Easy to Use Interface

Simple and clean design so users can quickly understand and use each tool without confusion.
